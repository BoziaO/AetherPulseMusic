# Contributing

Thanks for improving AetherPulse|Music.

## Workflow

1. Create a focused branch.
2. Keep changes small and easy to review.
3. Run the app with `npm run dev`.
4. Update `README.md` when behavior, setup, or features change.

## Code style

- Prefer clear React components and existing project patterns.
- Keep API calls in `src/lib/api.js` or backend routes.
- Use relative frontend API paths unless a deployment requires otherwise.
- Do not commit secrets, `.env`, `headers.json`, or generated builds.
